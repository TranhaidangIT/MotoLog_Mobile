# MotoLog – V6: Gộp dữ liệu Bảo dưỡng + Nhắc lịch, thêm màn "Thêm nhắc lịch"

> Sửa lỗi kiến trúc: Bảo dưỡng và Nhắc lịch hiện đang là 2 list dữ liệu cứng riêng biệt, lặp lại cùng thông tin. File này gộp về **1 nguồn dữ liệu duy nhất**, tính "còn bao nhiêu km" tự động từ ODO thật, và bổ sung màn hình "Thêm nhắc lịch" cho các nhắc **không** liên quan bảo dưỡng (đổ xăng, đăng kiểm, bảo hiểm...).

---

## 1. Nguyên tắc thiết kế lại

| Trước (sai) | Sau (đúng) |
|---|---|
| `maintenance_screen.dart` có data cứng riêng | Cả 2 màn hình đọc từ **1 danh sách `MaintenanceItem`** duy nhất |
| `reminder_screen.dart` có data cứng riêng, lặp lại y hệt | Mục "Định kỳ" trong Nhắc lịch = tự lọc ra từ `MaintenanceItem` có `isReminderOn = true` |
| "Còn X km" gán cứng | "Còn X km" = `interval - (currentOdo - lastDoneOdo)`, tính tự động |
| Không có nơi tạo nhắc không-liên-quan-bảo-dưỡng | Thêm `AddReminderScreen` cho nhắc tuỳ ý (đổ xăng theo bình, đăng kiểm, bảo hiểm...) |
| Ghi nhận bảo dưỡng không ảnh hưởng Nhắc lịch | Lưu `AddMaintenanceScreen` → tự cập nhật `lastDoneOdo` của `MaintenanceItem` tương ứng → cả 2 màn hình tự refresh |

## 2. Model dữ liệu chung

### 2.1 `MaintenanceItem` — hạng mục bảo dưỡng định kỳ (dùng chung cho cả 2 màn hình)

File: `lib/models/maintenance_item.dart`

```dart
import 'package:flutter/material.dart';

class MaintenanceItem {
  final String id;
  final String name;            // "Thay nhớt máy"
  final IconData icon;
  final int intervalKm;         // chu kỳ, ví dụ 2000
  final int lastDoneOdo;        // ODO lúc thay gần nhất
  final bool isReminderOn;      // có hiện trong mục "Định kỳ" của Nhắc lịch không

  MaintenanceItem({
    required this.id,
    required this.name,
    required this.icon,
    required this.intervalKm,
    required this.lastDoneOdo,
    this.isReminderOn = true,
  });

  // Tính còn bao nhiêu km dựa trên ODO hiện tại của xe
  int remainingKm(int currentOdo) {
    final used = currentOdo - lastDoneOdo;
    final remaining = intervalKm - used;
    return remaining < 0 ? 0 : remaining;
  }

  double progress(int currentOdo) {
    final used = currentOdo - lastDoneOdo;
    final p = used / intervalKm;
    return p.clamp(0.0, 1.0);
  }

  // Phân loại mức độ khẩn cấp để hiện badge ở Nhắc lịch
  // "urgent" = còn < 5% chu kỳ, "soon" = còn < 15%, "normal" = còn lại nhiều
  String urgency(int currentOdo) {
    final p = progress(currentOdo);
    if (p >= 0.95) return 'urgent';
    if (p >= 0.85) return 'soon';
    return 'normal';
  }

  MaintenanceItem copyWith({int? lastDoneOdo, bool? isReminderOn}) {
    return MaintenanceItem(
      id: id, name: name, icon: icon, intervalKm: intervalKm,
      lastDoneOdo: lastDoneOdo ?? this.lastDoneOdo,
      isReminderOn: isReminderOn ?? this.isReminderOn,
    );
  }
}
```

### 2.3 Nguồn dữ liệu chung — đơn giản hoá bằng 1 Provider/Controller tạm (chưa cần database thật)

File: `lib/data/maintenance_store.dart` (đơn giản hoá — sau này thay bằng SharedPreferences/SQLite/Hive thật)

```dart
import 'package:flutter/material.dart';
import '../models/maintenance_item.dart';
import '../models/custom_reminder.dart';

class MaintenanceStore extends ChangeNotifier {
  static final MaintenanceStore instance = MaintenanceStore._();
  MaintenanceStore._();

  int currentOdo = 15200; // TODO: lấy từ MyVehicleScreen / hồ sơ xe thật

  final List<MaintenanceItem> items = [
    MaintenanceItem(id: 'oil', name: 'Thay nhớt máy', icon: Icons.opacity, intervalKm: 2000, lastDoneOdo: 14700),
    MaintenanceItem(id: 'cvt', name: 'Vệ sinh nồi (CVT)', icon: Icons.settings, intervalKm: 10000, lastDoneOdo: 12900),
    MaintenanceItem(id: 'spark', name: 'Thay bugi', icon: Icons.electrical_services, intervalKm: 10000, lastDoneOdo: 12900),
    MaintenanceItem(id: 'air', name: 'Thay lọc gió', icon: Icons.air, intervalKm: 12000, lastDoneOdo: 10900),
    MaintenanceItem(id: 'coolant', name: 'Thay nước làm mát', icon: Icons.water_drop, intervalKm: 15000, lastDoneOdo: 8900),
    MaintenanceItem(id: 'battery', name: 'Thay ắc quy', icon: Icons.battery_full, intervalKm: 20000, lastDoneOdo: 11700),
  ];

  final List<CustomReminder> customReminders = [
    CustomReminder(id: 'fuel', title: 'Nhắc đổ xăng', subtitle: 'Khi còn dưới 1/4 bình', type: ReminderType.byFuelLevel),
  ];

  // Gọi khi người dùng lưu "Thêm bảo dưỡng" cho 1 hạng mục đã có sẵn
  void markDone(String itemId, int odoAtDone) {
    final i = items.indexWhere((e) => e.id == itemId);
    if (i == -1) return;
    items[i] = items[i].copyWith(lastDoneOdo: odoAtDone);
    notifyListeners();
  }

  void addCustomItem(MaintenanceItem item) {
    items.add(item);
    notifyListeners();
  }

  void toggleReminder(String itemId, bool value) {
    final i = items.indexWhere((e) => e.id == itemId);
    if (i == -1) return;
    items[i] = items[i].copyWith(isReminderOn: value);
    notifyListeners();
  }

  void addCustomReminder(CustomReminder reminder) {
    customReminders.add(reminder);
    notifyListeners();
  }
}
```

> Đây là bản đơn giản dùng `ChangeNotifier` singleton để 2 màn hình tự đồng bộ ngay, không cần setup `provider` package phức tạp. Khi nâng cấp lên lưu trữ thật (SQLite/Hive/Supabase), chỉ cần thay phần đọc/ghi bên trong class này, các màn hình không cần đổi.

## 4. Sửa `add_maintenance_screen.dart` — chọn hạng mục có sẵn, KHÔNG nhập tay khoảng cách km

Thay dropdown "Hạng mục bảo dưỡng" để load từ `MaintenanceStore.instance.items` (tên), và khi lưu thì gọi `markDone()`:

```dart
import '../data/maintenance_store.dart';

// Dropdown lấy items thật:
DropdownButtonFormField<String>(
  value: _selectedItemId,
  decoration: _decoration('Chọn hạng mục'),
  items: MaintenanceStore.instance.items
      .map((m) => DropdownMenuItem(value: m.id, child: Text(m.name)))
      .toList(),
  onChanged: (v) => setState(() => _selectedItemId = v),
),

// Khi bấm "Lưu lại":
onPressed: () {
  final odo = int.tryParse(_odoCtrl.text) ?? MaintenanceStore.instance.currentOdo;
  if (_selectedItemId != null) {
    MaintenanceStore.instance.markDone(_selectedItemId!, odo);
  }
  // TODO: lưu thêm chi phí, ghi chú vào lịch sử bảo dưỡng (HistoryScreen)

```

> Kết quả: ghi nhận "Thay bugi ở ODO 16.000" → `MaintenanceItem` của "Thay bugi" tự reset `lastDoneOdo = 16000` → cả màn Bảo dưỡng và mục "Định kỳ" ở Nhắc lịch tự cập nhật lại số km còn lại, không cần sửa 2 nơi.

## 5. Sửa `reminder_screen.dart` — mục "Định kỳ" tự sinh từ `MaintenanceStore`, không còn data cứng

```dart
import '../data/maintenance_store.dart';
import 'add_reminder_screen.dart';

// Trong build(), dùng ListenableBuilder để tự refresh:
ListenableBuilder(
  listenable: MaintenanceStore.instance,
  builder: (context, _) {
    final store = MaintenanceStore.instance;
    final currentOdo = store.currentOdo;

    // Lọc các hạng mục có isReminderOn = true để hiện ở "Định kỳ"
    final periodicItems = store.items.where((i) => i.isReminderOn).toList();

    // Phân loại "Sắp tới" = urgency != normal, sắp xếp theo % gần nhất
    final upcoming = periodicItems.where((i) => i.urgency(currentOdo) != 'normal').toList()
      ..sort((a, b) => a.remainingKm(currentOdo).compareTo(b.remainingKm(currentOdo)));

    final periodic = periodicItems.where((i) => i.urgency(currentOdo) == 'normal').toList();

    return Scaffold(
      // ... render 2 section "SẮP TỚI" (upcoming + customReminders sắp tới) và "ĐỊNH KỲ" (periodic)
      // Badge màu theo urgency: 'urgent' = đỏ "Gần tới", 'soon' = cam "Hôm nay"/"Sắp tới", 'normal' = xanh "Bình thường"
    );
  },
)
```

**Phần "SẮP TỚI"** gồm 2 nguồn trộn lại: hạng mục bảo dưỡng sắp đến hạn (`upcoming`) + `CustomReminder` nào đang `isOn = true` và đã/sắp đến hạn (ví dụ "Nhắc đổ xăng").

**Nút "+ Thêm nhắc lịch"** ở cuối màn hình → mở `AddReminderScreen` (mục 6), KHÔNG mở `AddMaintenanceScreen` — vì 2 cái khác mục đích:
- "Thêm bảo dưỡng" = ghi nhận đã làm 1 việc bảo dưỡng (cập nhật tiến độ)
- "Thêm nhắc lịch" = tạo 1 lời nhắc MỚI không liên quan chu kỳ bảo dưỡng có sẵn

## 6. Màn hình mới — `AddReminderScreen` (Thêm nhắc lịch tuỳ ý)

File: `lib/screens/add_reminder_screen.dart`

Dùng cho các nhắc KHÔNG gắn với hạng mục bảo dưỡng: đăng kiểm, bảo hiểm, đổ xăng theo mức bình, đóng thuế đường bộ, hẹn riêng...

Bố cục: chọn loại nhắc (segmented: Theo ngày / Theo km / Theo mức xăng) → form tương ứng đổi theo loại → tên nhắc → nút Lưu.

```dart
import 'package:flutter/material.dart';
import 'package:google_fonts/google_fonts.dart';
import '../theme/app_theme.dart';
import '../data/maintenance_store.dart';
import '../models/custom_reminder.dart';

class AddReminderScreen extends StatefulWidget {
  const AddReminderScreen({super.key});
  @override
  State<AddReminderScreen> createState() => _AddReminderScreenState();
}

class _AddReminderScreenState extends State<AddReminderScreen> {
  ReminderType _type = ReminderType.byDate;
  final _titleCtrl = TextEditingController();
  DateTime _date = DateTime.now();
  final _kmCtrl = TextEditingController();
  double _fuelFraction = 0.25; // 1/4 bình

  static const _typeLabels = {
    ReminderType.byDate: 'Theo ngày',
    ReminderType.byKm: 'Theo km',
    ReminderType.byFuelLevel: 'Theo mức xăng',
  };

  InputDecoration _decoration(String hint) => InputDecoration(
    hintText: hint,
    hintStyle: GoogleFonts.beVietnamPro(fontSize: 13, color: AppColors.textSecondary),
    filled: true,
    fillColor: Colors.white,
    contentPadding: const EdgeInsets.symmetric(horizontal: 14, vertical: 14),
    border: OutlineInputBorder(borderRadius: BorderRadius.circular(10), borderSide: BorderSide(color: AppColors.divider)),
    enabledBorder: OutlineInputBorder(borderRadius: BorderRadius.circular(10), borderSide: BorderSide(color: AppColors.divider)),
  );

  Widget _label(String text) => Padding(
    padding: const EdgeInsets.only(bottom: 6, top: 14),
    child: Text(text, style: GoogleFonts.beVietnamPro(fontSize: 12, color: AppColors.textSecondary)),
  );

  Widget _buildTypeSpecificField() {
    switch (_type) {
      case ReminderType.byDate:
        return TextField(
          readOnly: true,
          controller: TextEditingController(text: '${_date.day.toString().padLeft(2, '0')}/${_date.month.toString().padLeft(2, '0')}/${_date.year}'),
          decoration: _decoration('Chọn ngày hết hạn').copyWith(suffixIcon: const Icon(Icons.calendar_today_outlined, size: 18)),
          onTap: () async {
            final picked = await showDatePicker(context: context, initialDate: _date, firstDate: DateTime(2020), lastDate: DateTime(2040));
            if (picked != null) setState(() => _date = picked);
          },
        );
      case ReminderType.byKm:
        return TextField(
          controller: _kmCtrl,
          keyboardType: TextInputType.number,
          decoration: _decoration('Nhắc khi đạt mốc km (ví dụ: 20000)'),
        );
      case ReminderType.byFuelLevel:
        return Column(crossAxisAlignment: CrossAxisAlignment.start, children: [
          Text('Nhắc khi bình xăng còn ${(_fuelFraction * 100).round()}%',
            style: GoogleFonts.beVietnamPro(fontSize: 13, fontWeight: FontWeight.w600)),
          Slider(
            value: _fuelFraction, min: 0.1, max: 0.5, divisions: 8,
            activeColor: AppColors.primary,
            onChanged: (v) => setState(() => _fuelFraction = v),
          ),
        ]);
    }
  }

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(title: const Text('Thêm nhắc lịch'), leading: const BackButton()),
      body: ListView(
        padding: const EdgeInsets.all(16),
        children: [
          _label('Loại nhắc lịch'),
          Row(children: ReminderType.values.map((t) => Expanded(
            child: GestureDetector(
              onTap: () => setState(() => _type = t),
              child: Container(
                margin: const EdgeInsets.only(right: 6),
                padding: const EdgeInsets.symmetric(vertical: 9),
                decoration: BoxDecoration(
                  color: _type == t ? AppColors.primary : Colors.white,
                  borderRadius: BorderRadius.circular(10),
                  border: Border.all(color: _type == t ? AppColors.primary : AppColors.divider),
                ),
                alignment: Alignment.center,
                child: Text(_typeLabels[t]!, style: GoogleFonts.beVietnamPro(
                  fontSize: 11, fontWeight: FontWeight.w600,
                  color: _type == t ? Colors.white : AppColors.textSecondary,
                )),
              ),
            ),
          )).toList()),

          _label('Tên nhắc lịch'),
          TextField(controller: _titleCtrl, decoration: _decoration('Ví dụ: Đăng kiểm, Bảo hiểm xe...')),

          _label(_type == ReminderType.byDate ? 'Ngày hết hạn'
                : _type == ReminderType.byKm ? 'Mốc km'
                : 'Ngưỡng cảnh báo'),
          _buildTypeSpecificField(),

          const SizedBox(height: 24),
          SizedBox(
            width: double.infinity,
            child: ElevatedButton(
              onPressed: () {
                final subtitle = switch (_type) {
                  ReminderType.byDate => 'Hết hạn: ${_date.day.toString().padLeft(2, '0')}/${_date.month.toString().padLeft(2, '0')}/${_date.year}',
                  ReminderType.byKm => 'Tại mốc ${_kmCtrl.text} km',
                  ReminderType.byFuelLevel => 'Khi còn dưới ${(_fuelFraction * 100).round()}% bình',
                };
                MaintenanceStore.instance.addCustomReminder(CustomReminder(
                  id: DateTime.now().millisecondsSinceEpoch.toString(),
                  title: _titleCtrl.text.isEmpty ? 'Nhắc lịch mới' : _titleCtrl.text,
                  subtitle: subtitle,
                  type: _type,
                ));
                Navigator.pop(context);
              },
              style: ElevatedButton.styleFrom(
                backgroundColor: AppColors.primary,
                padding: const EdgeInsets.symmetric(vertical: 14),
                shape: RoundedRectangleBorder(borderRadius: BorderRadius.circular(12)),
              ),
              child: Text('Lưu lại', style: GoogleFonts.beVietnamPro(fontSize: 15, fontWeight: FontWeight.w600, color: Colors.white)),
            ),
          ),
        ],
      ),
    );
  }
}
```

## 7. Sơ đồ luồng dữ liệu (để AI hiểu rõ quan hệ)

```
MaintenanceStore (nguồn dữ liệu duy nhất)
│
├── items: List<MaintenanceItem>  ──┬──> MaintenanceScreen (hiện tất cả, progress bar)
│                                    └──> ReminderScreen mục "ĐỊNH KỲ" (chỉ item isReminderOn=true)
│                                          + mục "SẮP TỚI" (item nào urgency != normal)
│
├── customReminders: List<CustomReminder> ──> ReminderScreen mục "SẮP TỚI" (trộn cùng items sắp tới)
│
AddMaintenanceScreen ──(markDone)──> cập nhật lastDoneOdo của 1 MaintenanceItem
                                       → cả MaintenanceScreen và ReminderScreen tự refresh

AddReminderScreen ──(addCustomReminder)──> thêm 1 CustomReminder mới
                                       → chỉ ảnh hưởng ReminderScreen, không liên quan MaintenanceScreen
```
