# Lời nhắc cho Antigravity — Màn hình Đổ Xăng MotoLog

---

## Ngữ cảnh dự án

Tôi đang xây dựng app Flutter tên **MotoLog** — ứng dụng quản lý xe máy (theo dõi, bảo dưỡng, tiết kiệm). Màn hình cần làm là **màn hình ghi nhật ký đổ xăng** được thiết kế lại hoàn toàn để giảm thao tác người dùng xuống tối thiểu.

**Theme màu:** Primary `#2E7D32` (xanh lá đậm), font `Be Vietnam Pro` (Google Fonts), Material 3.

## Yêu cầu màn hình `FuelLogScreen`

### Mục tiêu UX
Người dùng chỉ cần nhập **2 trường** — số lít và ODO. Mọi thứ còn lại tự động.

### Các tính năng cần implement

#### 1. Tự động detect địa điểm qua GPS
- Khi mở màn hình, tự gọi `geolocator` lấy tọa độ hiện tại
- Query **Overpass API** (OpenStreetMap, miễn phí) trong bán kính 200m để tìm cây xăng gần nhất:
  ```
  GET https://overpass-api.de/api/interpreter
  Body: [out:json];node["amenity"="fuel"](around:200,{lat},{lon});out;
  ```
- Hiện tên cây xăng tìm được trong ô địa điểm (dạng read-only có thể bấm chọn lại)
- Nếu không tìm thấy → hiện "Không tìm thấy cây xăng gần đây" và cho phép nhập tay
- Nếu tìm thấy nhiều → hiện bottom sheet danh sách để chọn
- Có nút refresh để detect lại

#### 2. Giá xăng tự động
- Có model `FuelPrice` lưu giá các loại xăng: RON 95-III, RON 95-IV, E5 RON 92, dầu diesel
- Hiện dropdown chọn loại xăng, sau khi chọn tự điền giá/lít
- Giá được hardcode tạm thời (sẽ fetch API sau), ví dụ:
  ```dart
  static const prices = {
    'RON 95-IV':  21310,
    'RON 95-III': 20800,
    'E5 RON 92':  20050,
    'Dầu diesel': 18950,
  };
  ```
- Ghi chú trong code: `// TODO: Fetch từ Firebase Remote Config`

#### 3. Tính tiền tự động
- Khi user nhập **số lít** → `tổng tiền = số lít × giá/lít` (tính realtime)
- Khi user nhập **số tiền** → `số lít = tổng tiền / giá/lít` (tính realtime)
- Hai ô này sync 2 chiều, chỉ edit 1 ô tại 1 thời điểm (dùng `FocusNode` để tránh loop)

#### 4. Tính toán mức tiêu hao
- Lấy ODO lần đổ trước từ local storage (hardcode `12000` km nếu chưa có)
- User nhập ODO hiện tại → `quãng đường = ODO_hiện - ODO_trước`
- `Mức tiêu hao = quãng đường / số_lít` (km/lít)
- Hiện real-time trong card "Tính toán" ở cuối form

#### 5. Nhắc lịch đổ xăng
- Sau khi lưu thành công, tính: `km còn lại ≈ mức_tiêu_hao × (dung_tích_bình - số_lít_vừa_đổ)`
- Hiện dialog: *"Dựa trên mức tiêu hao của bạn, bạn cần đổ xăng sau khoảng X km nữa. Bật nhắc lịch không?"*
- Nếu đồng ý → tạo local notification (dùng `flutter_local_notifications`)
- Ghi chú: `// TODO: Tính từ ODO thực tế khi có sensor`

### Yêu cầu code style

- Dùng **StatefulWidget** cho màn hình chính
- Tách widget nhỏ thành private class bên dưới (không tạo file riêng trừ khi được yêu cầu)
- Mọi màu dùng `AppColors.primary` (`const Color(0xFF2E7D32)`)
- Loading state dùng `CircularProgressIndicator` màu xanh lá
- Error state hiện `SnackBar` đỏ
- Tất cả số tiền format bằng `NumberFormat('#,###', 'vi_VN')` từ package `intl`
- Animation dùng `flutter_animate` (fadeIn, slideY)
- Không dùng `GetX`, `Riverpod`, hay bất kỳ state management nào — chỉ `setState`

### Ví dụ call Overpass API

```dart
Future<List<GasStation>> findNearbyGasStations(
  double lat, double lon, {int radius = 200}
) async {
  final query = '''
[out:json];
node["amenity"="fuel"](around:$radius,$lat,$lon);
out;
  ''';
  final response = await http.post(
    Uri.parse('https://overpass-api.de/api/interpreter'),
    body: query,
  );
  final data = jsonDecode(response.body);
  return (data['elements'] as List).map((e) => GasStation(
    name: e['tags']['name'] ?? e['tags']['brand'] ?? 'Cây xăng',
    lat: e['lat'],
    lon: e['lon'],
  )).toList();
}
```

### Luồng hoạt động khi mở màn hình

```
initState()
  ├─ Set ngày = hôm nay
  ├─ Load ODO lần trước từ SharedPreferences
  ├─ Load giá xăng (hardcode, loại mặc định RON 95-III)
  └─ Gọi _detectLocation()
       ├─ Request permission (nếu chưa có)
       ├─ getCurrentPosition()
       └─ findNearbyGasStations()
            ├─ Tìm thấy 1 → tự điền
            ├─ Tìm thấy nhiều → hiện bottom sheet chọn
            └─ Không thấy → để trống, enable nhập tay
```

### Lưu ý quan trọng

- **Không cần backend** — toàn bộ lưu local bằng `SharedPreferences` ở giai đoạn này
- GPS detect chạy **không block UI** — form vẫn dùng được trong khi đang tìm
- Overpass API đôi khi chậm (~1-2s) — hiện skeleton/loading trong ô địa điểm
- Giá xăng hardcode là **giá tạm**, để comment `// TODO: fetch Petrolimex` rõ ràng
- Toàn bộ text tiếng Việt, không có i18n
